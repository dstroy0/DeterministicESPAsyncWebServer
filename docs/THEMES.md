# Themes

> Gallery generated by `src/web/wizard/gen_themes.py gallery`; previews by `tools/render_theme_previews.cjs`. Do not edit by hand.

**68 themes.** Opt a served page into one with a build directive - `<!--#theme dracula-->` or `<!--#theme generated/dracula-->` before `</head>` (see [src/web/README.md](../src/web/README.md)). Need an exact color that is not here? Generate your own from one anchor color:

```sh
python src/web/wizard/gen_themes.py custom --base '#36454f' --mode dark --name charcoal --out my.css
```
